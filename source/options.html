Xdebug helper

One-click debugging, profiling and tracing

Introduction

First off install and configure Xdebug, after that your correct IDE key blow. Now start debugging, profiling or tracing by clicking the little bug in the addressbar:

Debugging, profiling & tracing disabled
Debugging enabled
Profiling enabled
Tracing enabled

IDE key

Select your IDE to use the default sessionkey or choose other to use a custom key.

Domain filter

By default we show the bug icon on all pages, but here you can whitelist domains where the icon should show up. Note that all filters are regular expressions.
